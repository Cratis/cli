# Cratis CLI

The Cratis CLI (`cratis`) is the official command-line tool for managing and exploring [Chronicle](https://cratis.io) event stores.
It connects to a Chronicle server and lets you inspect events, manage observers, browse read models, diagnose problems, and administer users and applications — all from the terminal.
